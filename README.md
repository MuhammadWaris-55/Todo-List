# Todo List App

A simple and user-friendly Todo List application built with **React** and **Context API**. This app allows users to add, edit, and delete todos easily. It also uses **localStorage** to save todos, so the data stays available even after refreshing or reopening the browser.

## Features

- Add new todos.
- Edit existing todos.
- Delete todos.
- Manage state using Context API.
- Save todos in localStorage.
- Persistent data after page refresh.

## Tech Stack

- React
- Context API
- JavaScript
- localStorage

## Description

This project is a basic Todo List App made with React. It uses the Context API for state management, which makes it easy to share todo data across components without passing props again and again. Users can create a todo, update it, and delete it whenever needed. The app also stores todos in localStorage, so the list remains saved even after the browser is refreshed.

## Getting Started

Follow these steps to run the project locally:

```bash
git clone https://github.com/MuhammadWaris-55/Todo-List.git
```

```
npm i
```

```
npm run dev
```
